## Introduction

A program that simulates a guessing game, which should generate a random number between 0 and 50. Request a terminal interface for the user to try to guess the number in up to 5 attempts. After each attempt, the program should inform whether the number entered by the user is greater or less than the generated number.

## Technologies used

- Java
- Spring Tools for Eclipse

## Tools

The <a href="https://www.oracle.com/br/java/technologies/downloads/" target="_blank"> Java</a> programming language was installed on Windows, version JDK 21.

The program IDE <a href="https://code.visualstudio.com/download" target="_blank"> Spring Tools for Eclipse</a> was installed, using the version 4.31.0.

## Running the project

1. **To get the repository, clone it:**

   ```bash
   git clone https://github.com/dornascarol/guessing-game.git
   cd guessing-game
   ```

2. **Click the "Run As" button in the Java Application to run the program:**
   ```
   Run As > Java Application
   ```
